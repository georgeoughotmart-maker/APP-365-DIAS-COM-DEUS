# 365 Dias com Deus

Um guia espiritual diário para fortalecer sua fé e conexão com Deus ao longo de um ano.

## 🚀 Como subir no GitHub

1. Crie um novo repositório no seu GitHub.
2. No seu terminal local, dentro da pasta do projeto:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
   git push -u origin main
   ```

## 🌐 Como implantar na Vercel

1. Vá para [vercel.com](https://vercel.com) e faça login com seu GitHub.
2. Clique em **"Add New"** > **"Project"**.
3. Importe o repositório que você acabou de criar.
4. A Vercel detectará automaticamente que é um projeto **Vite**.
5. Se você estiver usando a API do Gemini no futuro, adicione a variável de ambiente `GEMINI_API_KEY` nas configurações do projeto na Vercel.
6. Clique em **"Deploy"**.

## 🛠️ Tecnologias Utilizadas

- React 19
- Vite
- Tailwind CSS 4
- Lucide React (Ícones)
- Motion (Animações)
- LocalStorage (Para salvar progresso e nome do usuário)

## ✨ Funcionalidades

- **Entrada Personalizada**: O app te chama pelo nome.
- **Jornada de 365 Dias**: Um versículo, reflexão e tarefa para cada dia do ano.
- **Exame de Consciência**: Espaço para autoavaliação diária.
- **Dinâmica do Dia**: Atividades práticas para meditação.
- **Desafio de Impacto**: Incentivo para abençoar outras pessoas.
- **Calendário Interativo**: Navegue facilmente por todos os dias do ano.
- **Progresso**: Acompanhe sua evolução na jornada.
